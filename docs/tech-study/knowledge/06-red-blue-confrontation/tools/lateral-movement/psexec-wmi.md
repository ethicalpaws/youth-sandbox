## PsExec
>PsExec 是 Sysinternals 套件中的一款轻量级远程执行工具，让你无需在目标机器上安装客户端，就能远程执行程序

下载地址：https://live.sysinternals.com/PsExec64.exe
### 核心特性
| 特性 | 说明 |
|------|------|
| 无需客户端 | 不需要在远程机器上预先安装任何软件 |
| 交互式 Shell | 可以在远程系统上获得一个交互式的命令行窗口 |
| 多种认证 | 支持明文密码、NTLM 哈希传递 |
| 权限切换 | 可以以 SYSTEM 权限或指定用户身份运行程序 |
| 自带微软签名 | 有微软数字签名，这使得它在一定程度上能绕过某些安全软件（当然，现在很多也会监控它） |

### PsExec 的工作原理
1. 建立连接：通过 SMB 协议（主要是 TCP 445 端口）连接到远程机器的 ADMIN$ 共享（默认指向 C:\Windows 目录）。
2. 上传服务：将 PSEXESVC.exe 上传到远程机器的 ADMIN$ 目录下。
3. 安装并启动服务：通过 Service Control Manager (SCM) 远程创建并启动名为 PSEXESVC 的服务。
4. 建立通信管道：服务启动后，会创建几个命名管道（Named Pipes），用于在本地和远程机器间传输命令的输入、输出和错误流，实现你看到的“交互式”效果
