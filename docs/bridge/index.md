## 🔁 每周工作流

1. **更新** `the_bridge.md` 中的当周记录（5分钟）
2. **填写** `weekly_brief.md`（发送给四位战友填写各自的简报部分）
3. **必要时更新** `milestone_summary.md`（重大里程碑时）
4. **窗口切换时**：发送 `milestone_summary.md` 和 `weekly_brief.md` 给新战友窗口
