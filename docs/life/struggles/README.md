# 📔 烦恼记录

> 一个带密码验证的私人烦恼记录本，用于记录内心的思考、矛盾与成长。

## 📁 目录结构

```
life/struggles/
├── source.md          # 源文件（你在这里写内容）
├── index.md           # 生成的 HTML 页面（自动生成，不要手动编辑）
├── update_trouble.py   # 生成脚本
└── README.md         # 本文件
```

## 🚀 快速开始

### 1. 添加新记录

编辑 `source.md`，按以下格式添加内容：

```markdown
---
id: 1
date: 2026-06-03
time: 19:34
title: 我究竟是什么样的人？
icon: 😔
question: 我在意别人的看法吗？
answer: yes

### 标题
正文内容...

---
id: 2
date: 2026-06-07
time: 12:34
title: 另一个烦恼
icon: 😮‍💨
question: 解锁问题？
answer: 答案

### 另一个标题
更多内容...
```

### 2. 生成页面

```bash
python update_trouble.py
```

### 3. 查看效果

用浏览器打开 `index.md` 即可。

---

## 📝 source.md 格式说明

### 必填字段

| 字段 | 说明 | 示例 |
|------|------|------|
| id | 唯一数字标识 | 1 |
| date | 日期（YYYY-MM-DD） | 2026-06-03 |
| time | 时间（HH:MM） | 19:34 |
| title | 标题 | 我究竟是什么样的人？ |
| icon | 表情图标 | 😔 |
| question | 解锁问题 | 我在意别人的看法吗？ |
| answer | 问题答案 | yes |

### 正文格式支持

| 语法 | 效果 |
|------|------|
| ### 标题 | 三级标题 |
| #### 标题 | 四级标题 |
| **加粗文字** | 加粗 |
| - 列表项 | 无序列表 |
| 1. 列表项 | 有序列表 |
| > 引用内容 | 引用块 |
| \`\`\`代码块\`\`\` | 代码块 |

### 记录分隔

每条记录之间用 `---` 单独一行分隔。

---

## 🔧 脚本说明

`update_trouble.py` 功能：

1. 读取 source.md 内容
2. 解析每条记录的元数据和正文
3. 将正文转换为 HTML
4. 生成带密码验证的完整 HTML 页面
5. 输出到 index.md

---

## ⚠️ 注意事项

- `index.md` 由脚本自动生成，不要手动编辑
- 每次修改 source.md 后，需要重新运行脚本
- 答案区分大小写，输入时需与 answer 字段完全一致

## 🐛 常见问题

**Q: 运行脚本后 index.md 没有更新？**

检查 source.md 格式是否正确：
- 字段和正文之间必须有空行
- 记录之间用 `---` 分隔
- 每条记录必须以 `id:` 开头

**Q: 解锁时提示答案错误？**
- 答案区分大小写
- 检查 answer 字段的值

**Q: 如何删除一条记录？**
- 直接从 source.md 中删除该记录，重新运行脚本即可。
