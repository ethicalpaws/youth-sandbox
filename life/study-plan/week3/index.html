    两篇WP可以是不同题型（Web/Pwn/Misc），重点是写出完整的解题思路和踩坑记录。

    SSTI实验¶

    PortSwigger有专门的SSTI实验，优先做 Basic SSTI 和 Basic SSTI with code review 两个。

    信息泄露¶

    这个在SRC中特别高频，重点整理：

    类型 检测方法 危害
    .git泄露 dirsearch扫描 源码泄露、配置泄露
    .env泄露 目录扫描 数据库密码、API密钥
    Swagger泄露 /swagger-ui.html API接口泄露
    JS接口泄露 浏览器F12 未授权接口
    备份文件 /www.zip / backup.rar 源码泄露
